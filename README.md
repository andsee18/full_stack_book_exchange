# BookExchange MVP
Платформа для обмена книгами между пользователями. Проект включает клиентскую часть (React) и серверную часть (Spring Boot), а также использует LocalStack для работы с S3 хранилищем.
## Как запустить проект
Для запуска вам потребуется установленный **Docker** и **Docker Compose**.
1. Перейдите в корневую директорию проекта (где находится файл `docker-compose.yml`).
2. Выполните команду для сборки и запуска всех контейнеров в фоновом режиме:
   ```bash
   docker compose up -d --build
   ```
## Доступные сервисы
После успешного запуска приложение будет доступно по следующим адресам:
- **Веб-сайт (Frontend):** [http://localhost](http://localhost)
- **API (Backend):** [http://localhost:5000](http://localhost:5000)
- **S3 Хранилище (LocalStack):** [http://localhost:4566](http://localhost:4566)
## Остановка приложения
Для остановки работы контейнеров выполните:
```bash
docker compose down
```
