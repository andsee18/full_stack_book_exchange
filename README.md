#  Book Exchange Platform (Backend & Frontend)

Этот репозиторий содержит полный код для платформы книжного обмена, включая бэкенд на **Node.js/Express** с базой данных **SQLite3** и фронтенд на **React/Vite**.

## Статус проекта: Завершена Лабораторная работа 4

Текущий статус проекта отражает полностью реализованную серверную часть (бэкенд) и готовность к интеграции с фронтендом (ЛР №5).

---

## Стек технологий

* **Бэкенд:** Node.js, Express
* **База данных:** SQLite3
* **Фронтенд:** React, Vite, CSS (стилизация завершена в ЛР №3)
* **Язык:** JavaScript

---

## Структура проекта

* `server.js`: Основной файл бэкенда. Содержит логику подключения к БД, инициализацию таблиц и все маршруты API.
* `book_exchange.db`: База данных SQLite3 с таблицами `users`, `books`, `exchanges`.
* `frontend/`: Папка с исходным кодом React-приложения.

---

## Реализованный функционал (ЛР №4)

В рамках Лабораторной работы №4 реализован и протестирован следующий функционал на стороне бэкенда:

### 1. Аутентификация и Пользователи (`users`)
* `POST /api/auth/register`: Регистрация нового пользователя.
* `POST /api/auth/login`: Вход в систему.
* `GET /api/users/:id`: Получение данных профиля пользователя.
* `PUT /api/users/:id`: Обновление профиля.

### 2. Книги и Каталог (`books`)
* `GET /api/books`: Получение списка всех доступных книг (каталог).
* `GET /api/books/:id`: Получение детальной информации о книге.
* `POST /api/books`: Добавление новой книги в каталог.
* `PUT /api/books/:id`: Редактирование информации о книге.
* `DELETE /api/books/:id`: Удаление книги.

### 3. Обмены (`exchanges`)
* `POST /api/exchanges`: Создание нового запроса на обмен.
* `GET /api/exchanges/user/:id`: Получение списка активных обменов пользователя.
* `PUT /api/exchanges/:id`: Обновление статуса обмена (`confirmed`, `rejected`).

## Инструкция по запуску

### 1. Запуск Бэкенда (API)

Для запуска сервера, который обслуживает все API-запросы:

1.  Установите зависимости (если еще не сделали):
    ```bash
    npm install
    ```
2.  Запустите сервер:
    ```bash
    node server.js
    ```
    Сервер будет доступен по адресу: `http://localhost:5000`

### 2. Запуск Фронтенда (UI)

Перейдите в папку `frontend` и запустите React-приложение:

1.  Перейдите в папку:
    ```bash
    cd frontend
    ```
2.  Установите зависимости фронтенда:
    ```bash
    npm install
    ```
3.  Запустите приложение:
    ```bash
    npm run dev
    ```
    Фронтенд будет доступен по адресу: `http://localhost:5173` (или другому порту Vite).
