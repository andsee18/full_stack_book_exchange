#  Проект: Сервис обмена книгами (Full-stack MVP)

## Обзор Проекта (ЛР №2)
Проект реализует минимально жизнеспособный продукт (MVP) для сервиса обмена книгами. Текущая контрольная точка фокусируется на развертывании серверной части.

### Используемые технологии:
* **Backend:** Node.js, Express.js
* **Контроль версий:** Git / GitHub

---

## Инструкции по запуску
Для запуска сервера необходимо иметь установленный Node.js (включает npm).

1.  **Клонирование репозитория:**
    ```bash
    git clone [ВАША_ССЫЛКА_НА_РЕПОЗИТОРИЙ]
    cd [НАЗВАНИЕ_ВАШЕЙ_ПАПКИ_С_ПРОЕКТОМ]
    ```
2.  **Установка зависимостей:**
    ```bash
    npm install
    ```
3.  **Запуск сервера в режиме разработки (с nodemon):**
    ```bash
    npm run dev
    ```
    Сервер будет доступен по адресу: `http://localhost:3000`

## Базовые API-эндпоинты (ЛР №2)
Для проверки используйте Postman/Thunder Client.

| Метод | Маршрут | Описание |
| :--- | :--- | :--- |
| **GET** | `/health` | **Health Check.** Проверка работоспособности. |
| **GET** | `/api/books` | Каталог всех книг. |
| **GET** | `/api/books/:id` | Просмотр подробной информации о книге. |
| **POST** | `/api/books` | Добавление новой книги (CRUD). |
| **POST** | `/api/auth/register` | Регистрация пользователя (заглушка). |
| **POST** | `/api/bookings` | Создание нового запроса на бронирование (заглушка). |
