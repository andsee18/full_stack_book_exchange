# Book Exchange Full Stack

Сервис для обмена книгами, реализованный на стеке **Spring Boot** (Backend) и **React** (Frontend).
Проект позволяет пользователям размещать свои книги, искать интересные экземпляры у других и совершать обмены.

---

## Технологический стек

### Backend
*   **Java 21**
*   **Spring Boot 3** (Web, Security, JDBC)
*   **SQLite** (База данных)
*   **JWT** (JSON Web Token) для аутентификации

### Frontend
*   **React** (Create React App)
*   **Axios** (HTTP клиент)
*   **React Router** (Маршрутизация)

---

## Функциональность

### Аутентификация и Безопасность (Лабораторная №2)
*   **JWT Auth**: Безопасный вход с использованием `Access Token` и `Refresh Token`.
*   **Token Rotation**: Защита от кражи токенов с автоматической ротацией Refresh токена.
*   **Secure Cookies**: Хранение Refresh токена в `HttpOnly` cookie.
*   **Interceptor**: Автоматическое обновление токена на клиенте без разлогинивания пользователя.

### Управление доступом (RBAC - Лабораторная №1)
*   **Роли**: `GUEST`, `USER`, `ADMIN`.
*   **Админ-панель**: Интерфейс для управления пользователями и назначения ролей.
*   **Защита маршрутов**: Приватные и ролевые маршруты (`RequireAuth`, `RequireRole`).

### Основные возможности
*   **Каталог книг**: Просмотр, поиск и фильтрация.
*   **Управление книгами**: Добавление, редактирование и удаление своих книг.
*   **Обмены**: Создание заявок на обмен, принятие или отклонение входящих заявок.
*   **Рейтинг**: Оценка пользователей после завершения обмена.
*   **Профиль**: Редактирование личных данных и загрузка аватара.

## Запуск проекта

### Backend

1.  Перейдите в папку бэкенда:
    ```bash
    cd bookexchange_backend
    ```
2.  Соберите проект (требуется Maven и JDK 21):
    ```bash
    mvn clean install -DskipTests
    ```
3.  Запустите сервер:
    ```bash
    java -jar target/backendjava-0.0.1-SNAPSHOT.jar
    ```
    *Сервер запустится на порту `5000`.*

### Frontend

1.  Перейдите в папку фронтенда:
    ```bash
    cd frontend
    ```
2.  Установите зависимости (требуется Node.js):
    ```bash
    npm install
    ```
3.  Запустите приложение:
    ```bash
    npm start
    ```
    *Приложение будет доступно по адресу `http://localhost:3000`.*

## API Endpoints

Базовый URL: `http://localhost:5000/api`

| Группа | Описание |
| :--- | :--- |
| `/auth` | Регистрация, вход, выход, обновление токена |
| `/users` | Профили пользователей, управление ролями (Admin) |
| `/books` | CRUD операции с книгами |
| `/exchange` | Управление заявками на обмен |
| `/ratings` | Оценки и отзывы |
